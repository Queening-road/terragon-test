产品文档

产品特性

  • 安全隔离环境: 基于 AWS 的 Firecracker技术，毫秒级启动独立虚拟机
  • Git 集成: 支持代码仓库拉取、分支管理、自动提交PR
  • 并行智能体: 同时启动多个智能体，分工合作完成复杂任务
  • 云端运行: 无需本地配置环境，所有操作在云端完成
  • 多模型支持: 集成第三方模型 Kimi、Qwen、GLM

使用流程

  1. 登录网站，接入模型
  2. 切换到 Dashboard 页面，选择模型
  3. 在输入框输入指令，任务列表新增执行记录
  4. 点击任务进入详情，查看AI生成过程
  5. AI生成完成可保存结果到本地
